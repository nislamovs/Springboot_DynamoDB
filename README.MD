# SpringBoot DynamoDB CRUD App 


### Build

- To build project and docker image: `./gradlew clean build docker`

or

- go to `./scripts` and run `./buildNrun.sh`

### Ports

|  Port  |  Description             |
| ------ | ------------------------ |
|  8080  | application api          |
|  7001  | dynamo db admin ui       |
